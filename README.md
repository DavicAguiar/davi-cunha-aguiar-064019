# Processo Seletivo SEPLAG/MT - Analista de TI (Sênior)
**Candidato:** Davi da Cunha Aguiar
**Perfil:** Engenheiro da Computação - Front-end

## Tecnologias Utilizadas
- React + TypeScript
- Tailwind CSS
- RxJS (BehaviorSubject para State Management)
- Docker

## Como Executar
1. Instale as dependências: `npm install`
2. Rode o projeto: `npm run dev`

## Decisões de Arquitetura
- Implementação do Padrão Facade para desacoplar a UI da lógica de negócio.
- Gerenciamento de estado reativo com RxJS para garantir escalabilidade.
